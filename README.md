# retail-sales-powerbi-dashboard
Retail sales data analysis and interactive dashboard built using Power BI to analyze revenue, product categories, and customer purchasing behavior.

# Retail Sales Analysis using Power BI
## Project Overview
This project analyzes retail transaction data to understand customer purchasing behavior, sales performance, and revenue distribution across different product categories and shopping malls.
An interactive dashboard was created using Power BI to transform raw sales data into meaningful business insights.

## Tools Used
* Power BI
* Power Query
* DAX
* Data Modeling

## Dataset Attributes
The dataset contains the following information:
* Customer ID
* Age
* Gender
* Product Category
* Shopping Mall
* Payment Method
* Quantity Purchased
* Purchase Date
* Price

## Data Preparation
Data cleaning and transformation were performed using Power Query.
Steps performed:
* Removed duplicate values
* Converted data types
* Created multiple tables for better data modeling
* Structured the dataset for dashboard analysis

## Data Modeling
The dataset was divided into multiple tables to demonstrate relational data modeling.
Relationships were created using common fields such as:
* Customer ID
* Product Category

## DAX Measures
Several key measures were created using DAX:
Total Revenue
Total Orders
Average Revenue per Customer

## Dashboard Visualizations
The Power BI dashboard includes:
* KPI cards showing total revenue, orders, and quantity sold
* Revenue distribution across product categories
* Sales performance by shopping mall
* Payment method analysis
* Monthly revenue trend analysis
* Interactive slicers for filtering data

## Key Insights
* Total revenue generated is approximately **251.51M** from around **99K orders**
* A total of **299K products** were sold
* The **Clothing category generates the highest revenue**
* Cash is the most commonly used payment method
* Mall of Istanbul generates the highest revenue
* Monthly revenue trends show seasonal variations
